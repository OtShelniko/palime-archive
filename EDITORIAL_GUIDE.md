# Palime Archive — Editorial Guide

Правила разметки статей (CPT `article`).

---

## 1. Таксономии: справочник

### section — Раздел

Основной раздел архива: `cinema`, `lit`, `music`, `art`.

| | |
|---|---|
| **Обязательно** | Всегда. Без раздела статья не попадёт в навигацию. |
| **Сколько** | Строго 1. |
| **Нельзя** | Ставить несколько разделов. Если статья пересекает разделы — выбрать основной. |

---

### article-type — Тип статьи

Формат материала: `author` (про автора), `work` (про произведение), `selection` (подборка).

| | |
|---|---|
| **Обязательно** | Всегда. Определяет шаблон карточки и логику фильтров. |
| **Сколько** | Строго 1. |
| **Нельзя** | Комбинировать типы. Статья — либо про автора, либо про произведение, либо подборка. |

---

### person — Персона

Имя автора, режиссёра, музыканта, художника, о котором идёт речь в статье.

| | |
|---|---|
| **Обязательно** | Для типов `author` и `work`. |
| **Опционально** | Для `selection` — если подборка вокруг конкретных персон. |
| **Сколько** | 1–3. Один основной + соавторы/связанные. |
| **Нельзя** | Указывать автора *статьи* (это поле WordPress Author). Указывать всех упомянутых — только ключевых. |

---

### theme — Тема / мотив

Смысловая ось материала. Предзаполненные значения: власть, смерть, память, одиночество, любовь, насилие, религия, безумие, абсурд, идентичность, тело, страх, война, искусство, мораль.

| | |
|---|---|
| **Обязательно** | Для типов `author` и `work`. |
| **Опционально** | Для `selection`. |
| **Сколько** | 1–3. Выбирать только те мотивы, которые статья *раскрывает*, а не упоминает. |
| **Нельзя** | Ставить 5+ тем «на всякий случай». Не путать с жанром (`genre`) и эпохой (`era`). |

---

### editorial-flag — Редакторская метка

Кураторская оценка материала: `выбор редакции`, `канон`, `точка входа`, `essential`, `спорное`, `недооценённое`.

| | |
|---|---|
| **Обязательно** | Нет. Ставится только осознанно. |
| **Опционально** | Всегда. Это привилегия редакции, а не обязанность автора. |
| **Сколько** | 0–2. |
| **Нельзя** | Ставить автору на свой текст. Ставить `канон` и `спорное` одновременно (противоречие). |

**Значения:**
- `выбор редакции` — лучшее за период, ставит главред
- `канон` — общепризнанная классика, бесспорное
- `точка входа` — идеальный первый текст для новичка в теме
- `essential` — обязательное к прочтению в контексте раздела
- `спорное` — намеренно провокационная или дискуссионная позиция
- `недооценённое` — незаслуженно забытое/игнорируемое

---

### era — Эпоха

Историко-культурный период, к которому относится *предмет* статьи (не дата публикации). Значения: античность, средневековье, возрождение, барокко, классицизм, романтизм, реализм, модернизм, авангард, постмодернизм, современность.

| | |
|---|---|
| **Обязательно** | Для типов `author` и `work`. |
| **Опционально** | Для `selection` — если подборка привязана к периоду. |
| **Сколько** | 1–2. Два допустимо, если автор/произведение на стыке эпох. |
| **Нельзя** | Путать с `theme`. Эпоха — *когда*, тема — *о чём*. Кафка = модернизм (эпоха) + абсурд (тема). |

---

### genre — Жанр

Жанр произведения или творчества автора (роман, нуар, джаз, инсталляция и т.д.).

| | |
|---|---|
| **Обязательно** | Для типа `work`. |
| **Опционально** | Для `author` и `selection`. |
| **Сколько** | 1–2. |

---

### country — Страна

Страна происхождения автора или произведения.

| | |
|---|---|
| **Обязательно** | Для типов `author` и `work`. |
| **Опционально** | Для `selection`. |
| **Сколько** | 1–2. |

---

### Узкоспециализированные таксономии

Ставятся **только при релевантности раздела**:

| Таксономия | Раздел | Когда ставить |
|---|---|---|
| `film-movement` | cinema | Если фильм/режиссёр принадлежит конкретному движению (неореализм, новая волна...) |
| `music-scene` | music | Если музыкант принадлежит субкультуре/сцене (панк, джаз, электроника...) |
| `art-period` | art | Если художник/произведение привязано к периоду ИЗО (импрессионизм, поп-арт...) |

Количество: 0–1. Не дублировать то, что уже указано в `era` или `genre`.

## 2. Правила по типам статей

### Тип `author` — статья про автора

Текст о творческой личности: биография, анализ наследия, портрет.

| Таксономия | Обязательно | Пример |
|---|---|---|
| `section` | да | `cinema` |
| `article-type` | да | `author` |
| `person` | да | `Андрей Тарковский` |
| `era` | да | `модернизм` |
| `theme` | да | `память`, `религия` |
| `country` | да | `Россия` |
| `genre` | желательно | `арт-хаус` |
| `editorial-flag` | нет | `канон` |
| `film-movement` | если релевантно | — |

**Пример готовой разметки:**
```
section:        cinema
article-type:   author
person:         Андрей Тарковский
theme:          память, религия
era:            модернизм
country:        Россия
genre:          арт-хаус
editorial-flag: канон
```

---

### Тип `work` — статья про произведение

Рецензия, разбор, эссе о конкретном произведении.

| Таксономия | Обязательно | Пример |
|---|---|---|
| `section` | да | `lit` |
| `article-type` | да | `work` |
| `person` | да | `Франц Кафка` |
| `era` | да | `модернизм` |
| `theme` | да | `абсурд`, `одиночество` |
| `country` | да | `Чехия` |
| `genre` | да | `роман` |
| `editorial-flag` | нет | `точка входа` |

**Пример готовой разметки:**
```
section:        lit
article-type:   work
person:         Франц Кафка
theme:          абсурд, одиночество
era:            модернизм
country:        Чехия
genre:          роман
editorial-flag: точка входа
```

---

### Тип `selection` — подборка

Тематическая подборка: «5 фильмов про...», «Главные альбомы жанра», «С чего начать...».

| Таксономия | Обязательно | Пример |
|---|---|---|
| `section` | да | `music` |
| `article-type` | да | `selection` |
| `person` | нет | — |
| `era` | нет | `современность` (если подборка про эпоху) |
| `theme` | желательно | `насилие` |
| `country` | нет | — |
| `genre` | желательно | `панк` |
| `editorial-flag` | нет | `выбор редакции` |

**Пример готовой разметки:**
```
section:        music
article-type:   selection
theme:          насилие
genre:          панк
era:            современность
editorial-flag: выбор редакции
```

## 3. Как различать похожие таксономии

### `theme` vs `editorial-flag`

| | `theme` | `editorial-flag` |
|---|---|---|
| **Что** | О чём текст | Как редакция оценивает текст |
| **Кто ставит** | Автор при публикации | Редакция после ревью |
| **Пример** | «любовь», «война» | «канон», «спорное» |
| **Вопрос-тест** | «Какие мотивы раскрывает статья?» | «Какое место статья занимает в архиве?» |

> `theme` — про содержание. `editorial-flag` — про значимость.

### `era` vs `theme`

| | `era` | `theme` |
|---|---|---|
| **Что** | Когда (исторический период) | О чём (смысловой мотив) |
| **Пример** | `романтизм` | `одиночество` |
| **Вопрос-тест** | «К какой эпохе относится предмет статьи?» | «Какую тему/проблему статья исследует?» |

> Байрон: `era: романтизм`, `theme: одиночество, страх`. Эпоха — контекст, тема — суть.

---

## 4. Чек-лист перед публикацией

- [ ] Указан ровно 1 `section`
- [ ] Указан ровно 1 `article-type`
- [ ] Для `author`/`work`: есть `person`, `era`, `theme`, `country`
- [ ] Для `work`: есть `genre`
- [ ] `theme` не больше 3
- [ ] `editorial-flag` — только если согласовано с редакцией
- [ ] `era` отражает эпоху предмета, а не дату написания статьи
- [ ] `person` — это герой статьи, а не автор статьи
